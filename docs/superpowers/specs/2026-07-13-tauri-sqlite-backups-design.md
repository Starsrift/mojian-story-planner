# Tauri + SQLite 本地存储与备份设计

## 目标

将墨笺从“浏览器 IndexedDB 是主数据源”升级为 Windows/macOS 一致的桌面应用：Tauri 负责桌面生命周期和后台任务，SQLite 作为唯一主存储，数据库及备份统一位于用户主目录的 `.mojian`。

## 用户可见行为

- 桌面应用中的所有作品、章节、人物、关系、时间线、伏笔和百科数据都直接写入 `~/.mojian/mojian.db`。
- 每次写操作成功后界面才更新，因此“SQLite 已保存”表示写入已经完成，而不是仅修改了内存状态。
- 应用首次启动且数据库为空、并检测到 `~/.mojian/latest.json` 时，自动尝试导入该镜像，保留原有作品 ID 和关联关系。
- 纯浏览器 IndexedDB 不与 Tauri WebView 共享存储。此类旧用户通过“旧版导出单作品 JSON → 桌面版导入”升级，发布说明不得宣称可自动读取浏览器数据库。
- 后台每小时检查一次备份状态；距离上次成功备份达到 48 小时时，在 `~/.mojian/backups/` 创建一个一致性的 SQLite 副本。
- 最多保留最近 30 份自动备份；清理只影响历史副本，不影响主数据库。
- 手动导出/导入单作品 JSON 的能力继续保留，便于分享和跨设备迁移。
- 浏览器 `npm run dev` 仅作为 UI 预览模式，继续使用 IndexedDB，并明确显示“浏览器预览”，避免用户误以为它正在写 SQLite。

## 架构

### 前端存储边界

`src/db/storage.ts` 是 Zustand 与持久层之间的唯一接口。Tauri 环境通过 `invoke` 调用 Rust 命令；纯浏览器环境回退到已有 Dexie 实现。组件和 store 不直接感知 SQL，也不直接调用 Dexie。

### SQLite 数据模型

SQLite 使用三张表：

- `projects`：作品 ID、更新时间和完整 JSON。
- `entities`：实体类型、实体 ID、作品 ID、排序值和完整 JSON；通过外键级联删除作品数据。
- `metadata`：保存 schema 版本、旧数据迁移状态和最近一次成功备份时间。

实体 JSON 保留现有 TypeScript 模型，避免在首次桌面化时同时引入大规模字段迁移。用于查询和完整性约束的字段单独建列。

### 桌面生命周期

Tauri 启动时创建 `.mojian` 和 `backups` 目录、初始化 SQLite schema、尝试旧数据迁移，并立即执行一次“到期检查”。随后后台线程每小时检查一次。关闭主窗口时应用隐藏到系统托盘，自动备份计时仍然有效；托盘提供“显示墨笺”和“退出”。应用注册 Windows/macOS 开机自启动，后台启动时不主动弹出窗口。

### 备份一致性

备份使用 SQLite Online Backup API，而不是直接复制正在写入的数据库文件。备份先写入临时文件，成功后原子重命名为 `mojian-backup-<UTC 时间>.sqlite3`。只有成功完成并通过完整性检查后，才更新 `last_backup_at` 并执行 30 份保留清理。

## 错误处理

- 参数缺少 `id`/`projectId`、实体类型不在白名单、JSON 序列化失败时，Rust 命令返回可读错误，前端写操作不更新内存状态。
- 旧镜像损坏时不覆盖或删除原文件；记录迁移错误并继续用空数据库启动。
- 自动备份失败时不更新最近备份时间，下一小时会重试；失败不会阻止正常编辑。
- 数据库初始化失败属于阻断性错误，Tauri 启动失败并保留底层错误信息用于诊断。

## 测试与验收

- Rust 单元测试覆盖 schema 初始化、实体 CRUD、作品级联删除、章节/角色关联数据的事务删除、旧镜像迁移、48 小时到期判断、备份恢复可读性和保留 30 份。
- TypeScript 测试覆盖存储命令映射、真实保存状态和同一实体的并发写入串行化，并验证浏览器预览仍可构建。
- `cargo test`、`npm run build` 和 `npm run tauri build -- --debug --no-bundle` 均成功后，才能宣称桌面存储实现完成。
- Windows/macOS 启动脚本都检查 Node、Rust 和 Cargo，并给出相同的数据目录说明。

## 范围边界

- 本次不实现云同步、多设备冲突解决、增量备份、备份加密或可配置备份周期。
- 主题偏好仍可保存在 WebView 的 `localStorage`；作品内容不得再依赖浏览器存储。
- 自动备份固定每 48 小时、保留 30 份；这些值先作为常量，后续再做设置页。
